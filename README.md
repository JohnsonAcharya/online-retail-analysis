# Online Retail Analysis with R

## Project Overview

This project demonstrates an end-to-end retail data analysis workflow using R.

The project will cover:

- Data Import
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Business KPIs
- Data Visualization
- Quarto Reporting
- Interactive Shiny Dashboard

## Tools

- R
- tidyverse
- ggplot2
- Quarto
- Shiny
- Git
