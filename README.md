# 📊 Online Retail Analysis with R

## 📌 Project Overview

This project follows a structured end-to-end data analytics pipeline:


- Data Import
- Data Understanding
- Data Cleaning
- Data Validation
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Business KPIs
- Data Visualization
- Quarto Reporting
- Interactive Shiny Dashboard


---

## 🎯 Goal

To transform raw retail transaction data into actionable business insights using a reproducible analytics workflow.
